Projet de Génération de Factures

Stage de 10 semaines – Association Compucraft

Le but de ce projet était de réaliser un site internet qui permettait de créer et générer une facture ou un devis en format PDF avec des données fournies par l'utilisateur. L'utilisateur devait pouvoir se connecter et s'inscrire sur le site. Le site comportait plusieurs pages, mais par exemple il y avait la page de coordonnées où l'utilisateur devait rentrer ses coordonnées et les modifier afin qu'elles soient utilisées pour générer la facture en PDF.

Dans la réalisation de ce site je me suis principalement occupé de la partie backend, que j'ai réalisée en utilisant Python avec le framework Flask, et j'ai utilisé PostgreSQL pour la base de données. Ce projet a été très formateur pour moi puisque j'ai appris à utiliser Python Flask pendant que je le réalisais et qu'il m'a permis d'utiliser GitHub pendant le projet.
