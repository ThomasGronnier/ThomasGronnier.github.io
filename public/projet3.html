Projet – Générateur de photo de profil anime

Application web pour créer facilement des avatars anime uniques

Ce projet personnel consiste en une application web développée en Node.js qui permet de générer en un clic une image de profil de style anime via une interface intuitive. À chaque requête, une API est sélectionnée aléatoirement parmi plusieurs sources spécialisées, offrant une grande variété d’images.

Le système intègre également un mécanisme de secours pour garantir la génération d’une image même en cas de défaillance d’une API. L’image générée est affichée instantanément à l’écran, prête à être téléchargée ou utilisée comme avatar.

Cette application met en valeur l’intégration d’APIs tierces dans un contexte concret, tout en offrant une expérience utilisateur fluide et variée.
