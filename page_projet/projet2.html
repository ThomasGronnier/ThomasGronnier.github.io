Projet Météo

J'ai réaliser ce projet lors de mon temps libre

But du projet

L'objectif de ce projet était d'utiliser l'API openweathermap qui est une API qui nous donne plein de données météorologique en temps réel comme la température, l'humidité, la températue max,min, etc ... . J'ai défini que la ville et les informations données seraient définie par une adresse IP que je localise pour avoir le nom d'une ville, et le nom de cette ville je la donne à l'API qui va se charger de m'envoyer les données météorologiques associés au nom de la ville. Le fond et l'image du site change également en fonction du temps qu'il fait dans la ville.

Techno utilisé et ce que j'ai appris

J'ai réaliser ce projet en Python avec le framework Flask pour le back et de l'HTML/CSS pour le front. Ce projet m'a permis de m'entrainer à utiliser une API ainsi qu'a traiter ses informations (qui étaient au format JSON), mais il m'as aussi permis de m'exercer au Python,HTML et CSS
