Projet pour l'association Compucraft

J'ai réalisé ce site web pendant mon stage de 10 semaine que j'ai fait à l'association Compucraft

But du projet

Le but de ce projet était de réaliser un site internet qui permettait de créer et génerer une facture ou un devis en format PDF avec des données fournies par l'utilisateur. L'utilisateur devait pouvoir se connecter et s'inscrire sur le site. Le site comportait plusieurs pages, mais par exemple il y avait la page de coordonnées ou l'utilisateur devait rentré ses coordonnées et les modifiées afin qu'elle soit utiliser pour générer la facture en PDF.

Techno utilisées et ce que j'ai appris

Dans la réalisation de ce site je me suis principalement occupé de la partie backend, que j'ai réalisé en utilisant Python avec le framework Flask, et j'ai utilisé PostgreSQL pour la base de données. Ce projet a était très formateur pour moi puisque j'ai appris à utiliser Python Flask pendant que je l'ai réalisé et qu'il m'a permis d'utilisé GitHub pendant le projet.
